Project Details

Create a music collection web application using bootstrap, php, MySQL.
The main use of the website will be to keep track of the user's music library and log play sessions.

Bonus content: If you want to you can make it so that you can keep track of student progress and make it so that the user can log the progress of each student after a class. (let me know if you are interested)

Essential Features of the Website:
  • View the music catalogue

    Users should be able to see the entire catalogue of pieces using a browse tab. Each entity will display essential information like: the name, composer and difficulty.

  • Sort/Search the music catalogue

    Users should be able to sort and search the information from different criteria

  • Label each piece as mastered, learning or plan to learn

    This information can be changed by the user using a form using an edit button.
    The label will be viewable on the music catalogue, if you don't want to do the bonus content.

  • Log and view play sessions

    In a different page tab, user can create play session logs to write down comments, what the user played that day, approx. duration, etc.

  • Insert pieces into a database using a form

    The piece can be added using a form where users can enter information such as: the title of the piece, composer, difficulty, etc. You can add more information if you would want.

  • Edit sond details

    The pieces can be editable using a form via an edit button on the piece.

Bonus Content Features:
  • View profiles of users and students

    Each profile will contain information about the person and tabs to view their personal list (containing only pieces that they have labeled as mastered, learning or planning to learn) and a list of all play sessions/class logs

  • Log student class

    User can create a writeup of the progress of the student, comments, what pieces were covered, etc for the class

Learning Objectives:
  • Front-End Development:

    Applying bootstrap, javascript and jQuery to create a web application

  • Working with databases:

    Learn how to user MySQL and Apache

  • Back-End Development:

    Learn how to use php to communicate information from the web application and database
